# spigot-tips

[![Actions Status](https://github.com/ekuinox/spigot-tips/workflows/SpigotTipsBuild/badge.svg)](https://github.com/ekuinox/spigot-tips/actions)

SpigotサーバにTipsというかメモ機能を追加するだけのプラグイン

## Usage

- 新しくTipsを追加する
    + `/tips create <title> <body>`

- Tipsを表示する
    + `/tips show <title>`
